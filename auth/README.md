# auth

The three-archetype authentication library — operator, brand customer and
Foundation-internal — wrapping the identity provider behind a stable interface
so application code never binds to the provider directly. Covers identity
verification, password hashing, TOTP-based 2FA and the RBAC framework.

**Status:** placeholder. Implemented in Stage 0 issue 16 (shared library) and
issues 17–19 (the per-archetype surfaces).
